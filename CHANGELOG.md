# Changelog

## 1.0.0

Initial public release.

- Multi-project tabs with session persistence
- Multi-session per project (Claude Code + terminal)
- Real-time session status indicators (working/idle/waiting)
- Auto-update with signed binaries
- Split pane view
- Prompt navigation in terminal history
- Session restore (resume/new)
- Supervised and autonomous modes
- Crash reporting with log collection
- Anonymous usage analytics (opt-in)
- Cross-platform: macOS (ARM64/x64), Windows (x64), Linux (x64)
